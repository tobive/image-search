Image Search API

How to use this image search api:

  • Put the search query after the url: https://image-search-tobive.herokuapp.com/api/imagesearch/
  • The result will be the first 10 result obtained from Google Custom Search API as a JSON object.
  • To put offset to the wanted results, put query "?offset=(number of offset)" after search query. for example: https://image-search-tobive.herokuapp.com/api/imagesearch/uemura%20rina?offset=46
  • To see the log of the latest searched query open: https://image-search-tobive.herokuapp.com/api/latest/imagesearch/
